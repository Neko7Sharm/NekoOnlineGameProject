# 🔊 Selestia Horizon Sound Effects Directory (SFX)

This directory is designated for storing all game sound effects (.mp3, .wav, .ogg).

## 📁 Recommended Subfolder Structure:

- `sfx/combat/`
  - `attack_slash.mp3`
  - `attack_bow.mp3`
  - `spell_fire.mp3`
  - `spell_heal.mp3`
  - `monster_hit.mp3`
  - `player_hit.mp3`
  - `victory_fanfare.mp3`

- `sfx/ui/`
  - `button_click.mp3`
  - `window_open.mp3`
  - `item_purchase.mp3`
  - `gold_coins.mp3`

- `sfx/movement/`
  - `footstep.mp3`
  - `teleport_warp.mp3`
  - `door_open.mp3`

- `sfx/environment/`
  - `shrine_pray.mp3`
  - `ambient_wind.mp3`
