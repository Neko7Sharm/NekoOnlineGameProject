# 🔊 Selestia Horizon Sound Effects Directory (SFX)

This directory is designated for storing all game sound effects (.mp3, .wav, .ogg).

## 📁 Recommended Subfolder Structure:

- `sfx/combat/` (attacks, hits, spells, fanfares)
- `sfx/ui/` (clicks, window toggles, purchases)
- `sfx/movement/` (footsteps, teleport warps, door sounds)
- `sfx/environment/` (shrine heals, ambient loops)
