# Roadmap

## v0.5 - Core Gameplay & Combat
- Complete tactical combat loop in Whispering Forest.
- Establish hit rolls, AC, damage types, and weapon properties.
- Local browser state persistence.
- Quest board system (10 quests max, 5-minute refresh).

## v0.6 - Crafting & Exploration
- Implement Mastery system (Gathering, Mining, Logging, Alchemy, Blacksmithing).
- Introduce material tags and Alchemy Table experimentation.
- Material Encyclopedia and Recipe Book separation.

## v0.7 - The Multiplayer Demo
- Transition local state to authoritative Supabase backend.
- Party system implementation.
- Real-time world chat and player visibility in town.

## v1.0 - Full Release
- Complete story integration.
- Advanced dungeons and boss mechanics.
- Full release polish and audio integration.
