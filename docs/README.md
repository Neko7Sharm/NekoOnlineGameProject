# Selestia Horizon Documentation

Welcome to the central documentation hub for Selestia Horizon! 

This folder contains all the official guidelines, lore, art directions, and technical rules that shape the development of the game. Organizing our documentation here ensures that human developers and AI assistants alike share the exact same understanding of the game's identity, mechanics, and standards.

## Documentation Structure

- **[GUIDELINES.md](./GUIDELINES.md)**: The core design pillars and goals of Selestia Horizon.
- **[ART_BIBLE.md](./ART_BIBLE.md)**: Visual identity, art direction, and design rules.
- **[WORLD_LORE.md](./WORLD_LORE.md)**: Story, world-building, and characters.
- **[GAMEPLAY_RULES.md](./GAMEPLAY_RULES.md)**: Core mechanics, combat, and systems.
- **[UI_GUIDE.md](./UI_GUIDE.md)**: User interface standards.
- **[AUDIO_BIBLE.md](./AUDIO_BIBLE.md)**: Music and sound direction.
- **[NAMING_GUIDE.md](./NAMING_GUIDE.md)**: Standards for naming conventions.
- **[STYLE_GUIDE.md](./STYLE_GUIDE.md)**: Standards for in-game text and dialogue tone.
- **[PROMPT_GUIDE.md](./PROMPT_GUIDE.md)**: Standardized AI prompts for asset generation.
- **[TECHNICAL_RULES.md](./TECHNICAL_RULES.md)**: Coding standards, file structures, and asset specs.
- **[ROADMAP.md](./ROADMAP.md)**: Development milestones and future plans.
